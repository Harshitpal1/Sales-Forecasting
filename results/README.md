# Results Directory

This directory contains analysis results and outputs from the sales forecasting models.

## Subdirectories

### visualizations/
Contains all generated plots and charts:
- Model comparison charts
- Prediction vs actual plots
- Residual analysis plots
- Feature importance charts
- Seasonality and holiday impact visualizations

### Files
- `model_comparison.csv`: Comprehensive comparison of all models with performance metrics

## Viewing Results

All visualizations are saved as high-resolution PNG files (300 DPI) suitable for presentations and reports.

The Jupyter notebook will automatically generate and display these visualizations when you run it.
